# VANTAGE SPORTS — Brand Assets

Official brand asset library for web, print, and distributor materials.

## Folder Structure

```
brand-assets/
├── README.md                 # This file
├── logos/
│   ├── primary/              # VANTAGE SPORTS wordmark & lockups
│   ├── partner/              # Manufacturing partner logos (INFIV — OEM only)
│   └── favicon/              # Web icons
├── colors/
│   └── brand-palette.md      # Navy, gold, white specifications
├── fonts/
│   └── .gitkeep              # Future brand typography files
├── product-images/
│   ├── shuttles/             # G4, G6, G7 product shots
│   └── equipment/            # Rackets, bags, apparel
├── marketing/
│   ├── social/               # Facebook, Instagram templates
│   └── print/                # Flyers, one-pagers, POS
└── videos/
    ├── factory/              # Manufacturing partner footage
    └── product/              # Product demos & testing
```

## Usage Rules

- **Primary brand:** VANTAGE SPORTS only on customer-facing materials
- **INFIV:** Use only in "Manufacturing Partner" or "OEM Partner" context
- **Do not** co-brand INFIV as equal to VANTAGE SPORTS on headers or hero sections

## Web Integration

- Public media served from `/public/media/` (see Media Library page)
- Source masters stored in `brand-assets/` (not committed until assets are ready)
