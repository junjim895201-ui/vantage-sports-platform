# Media Library — Public Assets

Place web-ready images and videos here for the Media Library page.

```
public/media/
├── factory/       # Factory & production photos
├── products/      # G4, G6, G7, equipment
├── events/        # Tournaments, club testing
└── videos/        # MP4 or linked embed placeholders
```

Files in this folder are served at `/media/{path}`.
